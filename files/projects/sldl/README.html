(Rust) Standard Library Definition Language

A language designed for use in libraries to define depedencies, outputs, version information and more. Specifically designed for Rust or Rust-based languages.

Uses the file extension .sldl.

Installation

Installation EXE can be found under releases. There is currently no official Linux or MacOS support.

Dependencies (as shown in Cargo.toml):

  • none

Support

Has libraries for the following languages: !--([Rust](https://logancammish.github.io/response))--

Built-in support for the following languages: FeO2, Rust, Lua, C/C++

Syntax example

OUT {
  FROM  "./example1.rs":
    #func print(string msg);
    #func println(string msg);
    #func printf(string data, tuple msg);. 
    & Take 3 functions &
  FROM "./example2.rs":
    #string DATA;. 
    & Take string "DATA" &
  TAKE "./example3.rs". 
    & Take all data &
  FROM NOWHERE:
    #string = "one two three".
    #number32 = 34.
    & Create your own variable &
}.
DEPEND { 
  "examplerepo"="0.0.1".
}.
INFO {
  version = "0.0.1".
  date = "02/03/2023".
  published = "02/03/2023".
  public_release = false.
}.

Utilizing interpreter::main() you should get the result (in your language):

{ ["OUT"] = {
    print(),
    println(),
    printf(),
    "ABC",
    ABC,
    "one two three",
    34
  },
  ["DEPEND"] = {
    ["examplerepo", "0.0.1"],
  },
  ["INFO"] = { 
    0.0.1, "02/03/2023", "02/03/2023", false
  }
}

Please note: INFO returns an array. The default array indexs are: 0 = version, 1 = date, 2 = published, 3 = public_release (+1 for Lua)
